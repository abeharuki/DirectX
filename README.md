# アプリケーション
今の作品を作ろと思った経緯は、人と一緒にゲームをするのも好きなのですが、
ドラゴンズドグマやドラクエ10のようなcpuと一緒に戦うゲームが、
仲間を自分が引き連れている感じがあって好きなので、就職作品としてドラクエ10の様な作品を作ろうと思い作っています。<br>
<br>
進捗は以下の通りです。
<br>

* 味方AIの実装(ある程度自分で動いてくれる)
* 敵AIの実装
* 各味方キャラごとのスキルの実装
* 味方AIをビヘイビアツリーを使用して管理
* 画面全体の絵作り(UI)
* エンカウント時の演出 など<br>
